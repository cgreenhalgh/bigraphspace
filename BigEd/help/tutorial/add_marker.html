Once the web-cam has been connected to the runtime, you can begin to configure the associations between markers and 3D models that will allow any markers recognized in the video feed produced by this web-cam to be augmented with 3D models. This involves two tasks:

  1. selecting the markers to use
  2. selecting the 3D models to appear on these markers
To select the markers to use:
  1. expand the AR markers entry in the Possible Things panel, revealing a list of markers (labelled "marker A", "marker B", "marker C", "marker D", etc.)
  2. choose two of these markers, and drag-and-drop them into the ARTECT Editor panel, thereby creating an icon that looks like the marker
  3. add a connection between each marker icon and the web-cam

Each of these markers can now be configured with a 3D model. To make this configuration:

  • expand the 3D Objects entry in the Possible Things panel
  • Drag-and-drop the cello 3D model into the ARTECT Editor panel
  • add a connection between the icon representing the model and the icon representing the marker

This configuration can now be tested by placing a physical print-out of both of these markers in the field of view of the webcam.
