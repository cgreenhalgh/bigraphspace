Required: Eclipse 3.3
With Additional Plugins:
  • Graphical Editor Framework 3.3
  • Web Standard Tools 2.0
  1. Import the Physical Configuration Project
    1. File->Import...
    2. Select 'Existing Projects into Workspace' in the General folder and click Next.
    3. For the root directory, browse to a parent directory of the folder containing the physical configuration manager.
    4. 'ect.equip.physconf' should appear in the Projects list. Make sure the checkbox next to it is ticked, and click 'Finish'
  2. Build the generated files from the ontology:
    1. double click on 'buildWrappers.xml'
    2. Set properties 'jena.home' to an installation of Jena 2.3 and 'java.tools.jar' to a suitable JDK location
    3. select the 'wrap' (default) task and 'Run as...' 'Ant Build'
  4. Select File->Export... from the menu
  5. Select 'Eclipse Product' from the Plugin Delevopment folder, and click Next.
  6. For the configuration, browse and select 'ect.equip.physconf/physconf.product'
  7. Select directory you want the program exported
  8. Finish! A fully built version of the Physical Configuration Manager should be in the directory you specified.
