The BigEd plugin is design to allow editing of Bigraphs. Currently it supports the editing of bigraph xml files. It is still early in the development of BigEd, and many features are missing or not working. Use at your own risk. Development is ongoing though, and hopefully you'll be able to see the potential of BigEd.

Required:
  • Eclipse 3.5
  • Graphical Editor Framework 3.5
Build Plugin:
  1. Import or Checkout the BigEd project into Eclipse
  2. Select File->Export... from the menu
  3. Select 'Deployable Plugins and Fragments' from the Plugin Development folder, and click Next.
  4. Check 'bigraph.biged'
  5. Select a directory where you want the plugin exported
  6. Finish! A fully built version of BigEd should be in the directory you specified.
Install:
  1. Just copy the created plugin jar file into the Eclipse plugins directory are restart eclipse.
